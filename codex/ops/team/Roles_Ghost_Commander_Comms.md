# Predator Team Structure
**3 Roles. One brain. One voice. One set of hands.**

This system is designed to reduce human error and prevent emotional betting.

---

## Role 1: Ghost (AI Strategist)
**Primary function:** decision engine.

### Responsibilities
- Track rolling memory (short + long)
- Detect clusters and confirm engagement criteria
- Enforce exposure caps and anti-overcommit rules
- Trigger Recovery Mode at 50% drawdown
- Trigger Profit Lock immediately when thresholds hit
- Issue commands in short, unambiguous formats

### Ghost Rules
- Never issue “maybe” commands
- Never chase losses
- Never allow cluster narrowing during stagnation
- If data is unclear → Cold Mode

---

## Role 2: Communication Director (Voice / Interpreter)
**Primary function:** clean transmission.

### Responsibilities
- Speak Ghost’s commands clearly and consistently
- Confirm commands before chips go down
- Call unit count + bet shape out loud
- Stop play if confusion exists

### Communication Director Rules
- No improvising
- No adding “personal bets”
- If a command is unclear → ask for repeat, do not guess

---

## Role 3: Commander (Observer + Executor)
**Primary function:** battlefield execution.

### Responsibilities
- Read outcomes accurately
- Place chips exactly as called
- Confirm placement matches the command
- Maintain calm pace and physical control
- Stop play instantly if the system integrity breaks

### Commander Rules
- If you misread a number → pause immediately and reset memory
- If the table pace is too fast → slow down or step away
- If emotions rise → stop session (automatic trigger)

---

## Standard Command Format (Recommended)
Commands should be short and structured:

**MODE:** Cold / Engage / Recovery  
**BET:** (type + target set
